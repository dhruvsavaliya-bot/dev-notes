# dev-notes 📝

A growing collection of coding tips, language notes, AI/LLM lessons,
trending open-source projects, and articles worth reading.

## Structure
- `coding-tips/` — practical programming tips
- `languages/` — language-specific notes and gotchas
- `ai/` — lessons from building with LLMs
- `trending-projects/` — open-source projects worth checking out
- `articles/` — reading list of great blog posts

Updated regularly via `add-note.py`.
